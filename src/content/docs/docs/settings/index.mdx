## Operations broadcast by this section

- `account_whitelist` — manage whitelist/blacklist membership.
- `account_upgrade` — upgrade to Lifetime Member.
- `account_create` — register a new account (LTM method).

`Blocked Users`, `Nodes`, `Configure Visuals`, `Theme Customizer` and `Per-Page Themes` are local-only and never broadcast.
