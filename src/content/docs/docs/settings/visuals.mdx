**Configure Visuals** tunes the app's appearance — animated waves, particles, aurora, blur and the IPFS gateway used to load off-chain content. All of it is local UI state; nothing is broadcast.

**Open in the app:** [Configure Visuals](/visuals.html).

## What it does

<CardGrid>
  <Card title="Background effects" icon="palette">
    Wave count/speed, aurora intensity, particles, blur.
  </Card>
  <Card title="Palettes" icon="star">
    Rainbow, ocean, sunset, mono or custom colors.
  </Card>
  <Card title="IPFS gateway" icon="database">
    Choose which gateway serves off-chain data.
  </Card>
</CardGrid>

## How to use

<Steps>

1. Open **Configure Visuals** from the homepage under *Settings*.

2. Adjust wave/particle/aurora/blur sliders and pick a palette.

3. Set your preferred IPFS gateway.

4. Changes apply immediately and persist locally.

</Steps>
