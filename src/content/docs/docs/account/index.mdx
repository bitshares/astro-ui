## Operations broadcast by this section

- `limit_order_cancel` — cancel an open order.
- `custom_authority_create` / `_update` / `_delete` — account authorities.
- `credit_deal_repay` — repay a credit deal.
- `vesting_balance_withdraw` — claim vested funds.
- `proposal_update` / `proposal_delete` — approve or reject proposals.

Everything else is a read-only viewer.
