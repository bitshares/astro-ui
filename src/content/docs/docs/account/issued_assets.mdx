**Issued Assets** lists everything an account has issued: user-issued assets, smartcoins, liquidity pools and NFT-like assets. It's a read-only explorer with JSON viewers and links; issuer-level actions (issue/reserve/update) are delegated to the issuer-actions component.

**Open in the app:** [Issued Assets](/issued_assets.html).

## What it does

<CardGrid>
  <Card title="Full inventory" icon="boxes">
    UIAs, MPAs, pools and more issued by the account.
  </Card>
  <Card title="Inspect" icon="document">
    View each asset's JSON and configuration.
  </Card>
  <Card title="Act (issuer)" icon="settings">
    When you're the issuer, manage supply and flags.
  </Card>
</CardGrid>

## How to use

<Steps>

1. Open **Issued Assets** from the homepage under *Account Overviews*.

2. Select the issuing account. Tabs separate UIAs, smartcoins, pools, etc.

3. Open an asset to view its details and JSON.

4. If you are the issuer, use the issuer actions to manage it (those actions broadcast `asset_issue`, `asset_reserve`, `asset_update`, etc., outside this viewer).

</Steps>

<Aside type="note" title="Viewer only">
  This page lists and inspects issued assets. Creating one is done via [Create UIA](/docs/asset-creation/create_uia.html) or [Create SmartCoin](/docs/asset-creation/create_smartcoin.html).
</Aside>

## Operations used

When you are the issuer, the issuer-actions panel lets you manage supply, fees, feed producers, and ownership. Each action broadcasts a different operation.

<Tabs>
  <TabItem label="asset_issue">

Issue new tokens to a target account:

```json
{
  "issuer": "1.2.x",
  "asset_to_issue": { "amount": 1000000, "asset_id": "1.3.x" },
  "issue_to_account": "1.2.x",
  "memo": null,
  "extensions": {}
}
```

| Field | Meaning |
| --- | --- |
| `issuer` | Your account (`1.2.x`). |
| `asset_to_issue` | Amount (pre-scaled by precision) and asset id. |
| `issue_to_account` | Recipient of the newly issued tokens. |

  </TabItem>
  <TabItem label="asset_reserve">

Burn (reserve) tokens by returning them to the issuer:

```json
{
  "payer": "1.2.x",
  "amount_to_reserve": { "amount": 500000, "asset_id": "1.3.x" },
  "extensions": {}
}
```

| Field | Meaning |
| --- | --- |
| `payer` | Account surrendering the tokens. |
| `amount_to_reserve` | Amount and asset to burn. |

  </TabItem>
  <TabItem label="asset_fund_fee_pool">

Fund the asset's fee pool from your account balance:

```json
{
  "from_account": "1.2.x",
  "asset_id": "1.3.x",
  "amount": 10000000,
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="asset_claim_pool">

Claim BTS from the asset's fee pool:

```json
{
  "issuer": "1.2.x",
  "asset_id": "1.3.x",
  "amount_to_claim": { "amount": 5000000, "asset_id": "1.3.0" },
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="asset_claim_fees">

Claim accumulated market fees in the asset itself:

```json
{
  "issuer": "1.2.x",
  "amount_to_claim": { "amount": 200000, "asset_id": "1.3.x" },
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="asset_update_feed_producers">

Set the list of accounts allowed to publish price feeds:

```json
{
  "issuer": "1.2.x",
  "asset_to_update": "1.3.x",
  "new_feed_producers": ["1.2.x", "1.2.x"]
}
```

  </TabItem>
  <TabItem label="asset_global_settle">

Globally settle (force-close) a market-pegged asset at a fixed price:

```json
{
  "issuer": "1.2.x",
  "asset_to_settle": "1.3.x",
  "settle_price": {
    "base":  { "amount": 100000, "asset_id": "1.3.x" },
    "quote": { "amount": 100000, "asset_id": "1.3.0" }
  },
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="asset_update_issuer">

Transfer issuer authority to another account:

```json
{
  "issuer": "1.2.x",
  "new_issuer": "1.2.x",
  "asset_to_update": "1.3.x",
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="override_transfer">

Recall (force-transfer) tokens from any holder back to the issuer:

```json
{
  "issuer": "1.2.x",
  "from": "1.2.x",
  "to": "1.2.x",
  "amount": { "amount": 100000, "asset_id": "1.3.x" },
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="liquidity_pool_delete">

Delete a liquidity pool you created:

```json
{
  "account": "1.2.x",
  "pool": "1.19.x",
  "extensions": {}
}
```

  </TabItem>
</Tabs>
