## How it works

Invoices are **off-chain** base58-encoded payloads (recipient, identifier, line items). Only payment is on-chain: [Pay Invoice](/docs/invoicing/pay_invoice.html) broadcasts a `transfer` per asset, with the invoice `identifier` in the memo.

The only operation in this section is `transfer`; everything else is browser-local.
