## What is BitShares Astro UI?

**BitShares Astro UI** is a web and desktop client for the BitShares decentralized exchange and smart-chain. It lets you trade assets on the DEX, borrow against collateral, issue your own tokens, vote in network governance, and manage every corner of your account — all from a single interface that talks to the blockchain through your own wallet (Beet), so your keys never leave your machine.

This is the **documentation** for that application. It explains what each screen does, how to use it, and — for the pages that perform on-chain actions — the exact BitShares **operation** the app builds, with example JSON you can study and learn from.

## How to use these docs

The navigation on the **left side** (the sidebar) lists every page, grouped by the same sections you see on the app's home screen: Exchanging Funds, Transferring Funds, Forms of Debt, Asset Creation, Account Overviews, Blockchain Overviews, Governance, Invoicing, and Settings.

<Aside type="tip" title="Start from the sidebar">
  Pick the section that matches what you want to do, then open the specific page. Each page follows the same layout: what it does, how to use it, and (where relevant) the operation it broadcasts.
</Aside>

Most pages also link back to the live application, and cross-link to related pages so you can follow a task end to end — for example from [Borrow](/docs/debt/borrow.html) to [SmartCoins](/docs/debt/smartcoins.html) to [Lend](/docs/debt/lend.html).

## A note on operations

BitShares actions are expressed as **operations** — typed objects that your wallet serializes, signs and broadcasts. Throughout these docs, the action pages show the real operation name(s) and a representative JSON object using placeholder object ids:

<Badge text="1.2.x" variant="note" /> account &nbsp; <Badge text="1.3.x" variant="note" /> asset &nbsp; <Badge text="1.7.x" variant="note" /> limit order &nbsp; <Badge text="1.6.x" variant="note" /> witness

The `x` stands for the numeric id of that object on the chain. Read-only pages (like the blockchain explorers) don't broadcast anything, so they simply describe what you can view.
