## Common operations

Most pages in this section build on a small set of BitShares operations:

- `limit_order_create` / `limit_order_update` / `limit_order_cancel` — the order book.
- `liquidity_pool_exchange` / `liquidity_pool_deposit` / `liquidity_pool_withdraw` — pools.
- `proposal_create` (wrapping `transfer`) — barter.
- `samet_fund_borrow` / `samet_fund_repay` — TFunds.

Each page documents its exact operation shapes with example JSON.
