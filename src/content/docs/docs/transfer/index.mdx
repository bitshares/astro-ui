## Common operations

- `transfer` — the universal send.
- `proposal_create` (wrapping `transfer`) — timed transfers.
- `withdraw_permission_create` / `_update` / `_claim` / `_delete` — delegated withdrawals.
- `htlc_create` / `htlc_redeem` / `htlc_extend` — conditional escrow.
- `vesting_balance_create` — time-locked balances.
- `transfer_to_blind` / `blind_transfer` / `transfer_from_blind` — confidential transfers.

Each page documents its exact operation shapes with example JSON.
