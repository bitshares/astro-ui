**HTLC** (Hashed Timelock Contract) lets you lock funds for a recipient who can only claim them by revealing a secret preimage that hashes to an agreed value — within a `claim_period_seconds` window. It's the building block for cross-chain atomic swaps and conditional escrow. If the secret isn't revealed in time, the sender can refund.

**Open in the app:** [HTLC](/htlc.html).

## What it does

<CardGrid>
  <Card title="Secret-locked" icon="key">
    Funds release only to whoever knows the preimage.
  </Card>
  <Card title="Time-locked" icon="clock">
    Auto-refund to sender after the claim period.
  </Card>
  <Card title="Atomic swaps" icon="swap">
    The primitive behind trustless cross-chain trades.
  </Card>
</CardGrid>

## How to use

<Steps>

1. Open **HTLC** from the homepage under *Transferring Funds*.

2. Enter the recipient, asset/amount, and a `preimage_hash` (with its hash algorithm cipher) and `preimage_size`.

3. Set the `claim_period_seconds` window.

4. Create it (`htlc_create`). Share the secret preimage with the recipient off-chain.

5. Recipient redeems with `htlc_redeem` (supplying the preimage) or you `htlc_extend` the window if needed.

</Steps>

<Aside type="note" title="Hash algorithm cipher">
  `preimage_hash` is a two-element array `[cipher, hexHash]`. `cipher` is `0` for RIPEMD160 or `1` for SHA256. The redeemer must supply the exact original preimage of that hash.
</Aside>

## Operations used

<Tabs>
  <TabItem label="htlc_create">

```json
{
  "from": "1.2.x",
  "to": "1.2.x",
  "amount": { "amount": 250000, "asset_id": "1.3.0" },
  "preimage_hash": [1, "a665a45920422f9d417e4867efdc4fb8a04a1f3fff1fa07e998e86f7bcabe64"],
  "preimage_size": 32,
  "claim_period_seconds": 86400,
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="htlc_redeem">

```json
{
  "htlc_id": "1.13.x",
  "redeemer": "1.2.x",
  "preimage": "<the secret preimage string>",
  "extensions": {}
}
```

  </TabItem>
  <TabItem label="htlc_extend">

```json
{
  "htlc_id": "1.13.x",
  "update_issuer": "1.2.x",
  "seconds_to_add": 86400,
  "extensions": {}
}
```

  </TabItem>
</Tabs>

| Field | Meaning |
| --- | --- |
| `from` / `to` | Sender / recipient accounts. |
| `amount` | Funds locked in the contract. |
| `preimage_hash` | `[cipher, hexHash]` the redeemer must match. |
| `preimage_size` | Byte length of the preimage. |
| `claim_period_seconds` | Window before auto-refund. |
| `htlc_id` | The `1.13.x` HTLC object id. |
