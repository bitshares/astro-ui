## Common operations

- `call_order_update` — open/adjust/close a margin debt position (borrow).
- `credit_offer_create` / `credit_offer_update` / `credit_offer_accept` — the credit marketplace.
- `samet_fund_create` / `samet_fund_update` / `samet_fund_delete` — fund management.

Each page documents its exact operation shapes with example JSON.
